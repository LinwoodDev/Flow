# Elenco degli errori

| Codice di errore | Codice di stato HTTP | Descrizione             |
| ---------------- | -------------------- | ----------------------- |
| `0`              | `404`                | Pagina non trovata      |
| `1`              | `404`                | Versione API non valida |
| `2`              | `403`                | Vietato                 |
| `3`              | `400`                | Richiesta errata        |
