# Fejl liste

| Fejl kode | HTTP status kode | Varebeskrivelse        |
| --------- | ---------------- | ---------------------- |
| `0`       | `404`            | Siden blev ikke fundet |
| `1`       | `404`            | Ugyldig api version    |
| `2`       | `403`            | Forbudt                |
| `3`       | `400`            | Forkert anmodning      |
