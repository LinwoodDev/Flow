title: "Byg din egen"
sidebar_position: 1
---

1. Installer git og flutter (beta)
2. Klon depotet
3. Navigér til app-mappen
4. Brug flagerværktøjet til at kompilere programmet
   * `flutter byg apk`
   * `flutter opbygge æblebundt`
   * `flutter opbygge web`
   * `flutter build linux`
   * `flutter bygge vinduer`
5. De kompilerede filer er i byggemappen
