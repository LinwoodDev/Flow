# Liste des erreurs

| Code d'erreur | Code de statut HTTP | Libellé                   |
| ------------- | ------------------- | ------------------------- |
| `0`           | `404`               | Page introuvable          |
| `1`           | `404`               | Version de l'api invalide |
| `2`           | `403`               | Interdit                  |
| `3`           | `400`               | Mauvaise requête          |
