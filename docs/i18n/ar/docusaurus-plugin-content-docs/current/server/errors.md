# قائمة الأخطاء

| رمز الخطأ | رمز حالة HTTP | الوصف                    |
| --------- | ------------- | ------------------------ |
| `0`       | `404`         | لم يتم العثور على الصفحة |
| `1`       | `404`         | إصدار api غير صالح       |
| `2`       | `403`         | ممنوع                    |
| `3`       | `400`         | طلب خاطئ                 |
