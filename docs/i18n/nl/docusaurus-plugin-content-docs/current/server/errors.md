# Fout lijst

| Foutmelding code | HTTP status code | Beschrijving         |
| ---------------- | ---------------- | -------------------- |
| `0`              | `404`            | Pagina niet gevonden |
| `1`              | `404`            | Ongeldige API versie |
| `2`              | `403`            | Verboden             |
| `3`              | `400`            | Slecht verzoek       |
