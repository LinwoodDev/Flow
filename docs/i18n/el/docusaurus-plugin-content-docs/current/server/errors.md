# Λίστα σφαλμάτων

| Κωδικός σφάλματος | Κωδικός κατάστασης HTTP | Περιγραφή            |
| ----------------- | ----------------------- | -------------------- |
| `0`               | `404`                   | Η σελίδα δεν βρέθηκε |
| `1`               | `404`                   | Μη έγκυρη έκδοση api |
| `2`               | `403`                   | Απαγορευμένο         |
| `3`               | `400`                   | Εσφαλμένο αίτημα     |
