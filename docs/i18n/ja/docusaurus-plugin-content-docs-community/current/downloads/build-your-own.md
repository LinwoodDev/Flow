---
title: "自分で作ってください"
sidebar_position: 1
---

1. git とフラッタをインストール (ベータ版)
2. リポジトリを複製
3. アプリのディレクトリに移動します
4. フラッターツールを使用してアプリケーションをコンパイルする
   * `Flutter build apk`
   * `flutter build appbundle`
   * `flutter build web`
   * `flutter build linux`
   * `フラッタビルドウィンドウ`
5. コンパイルされたファイルはビルドディレクトリにあります
