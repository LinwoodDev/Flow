## Windows SmartScreen


Windowsは証明書なしでソフトウェアをインストールするときにユーザーに警告します。

![Smart screen](/img/smart-screen.png)

アプリをインストールするには、"More info"をクリックする必要があります。

![Smart screen more info](/img/smart-screen-more-info.png)

それから、「とにかく実行」をクリックしてください。
