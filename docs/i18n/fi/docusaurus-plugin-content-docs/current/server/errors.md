# Virheiden luettelo

| Virhekoodi | HTTP tilakoodi | Kuvaus                  |
| ---------- | -------------- | ----------------------- |
| `0`        | `404`          | Sivua ei löydy          |
| `1`        | `404`          | Virheellinen api versio |
| `2`        | `403`          | Kielletty               |
| `3`        | `400`          | Virheellinen pyyntö     |
