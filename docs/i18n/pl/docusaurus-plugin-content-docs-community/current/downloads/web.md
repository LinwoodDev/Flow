## Wersje

Główne i podgląd (rozwijają) wersji Flow nie są takie same jak wersje stabilne i nocne. Nie są one bezpośrednio związane z wydaniem. Są one aktualizowane natychmiast po wypychaniu nowego zobowiązania do repozytorium.
